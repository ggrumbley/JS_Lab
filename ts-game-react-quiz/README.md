# Quiz Game

✅ A little quiz game built in React ✅

## Technologies Used

+ Next.js
+ Typescript
+ Supabase
+ Tailwind CSS
+ PostCSS

## Quickstart

1. Create a Supabase Account
2. Create a Supabase Project
3. Run the Todo List SQL Quickstart
4. Copy over supabase project url and anon key to a `.env.local` file.
5. RUN `yarn && yarn dev`
6. Profit $$
